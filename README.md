## Executive Summary

A Cisco VOIP phone at the front counter lost network connectivity while continuing to receive power via Power over Ethernet (PoE). Troubleshooting isolated the fault to a damaged Ethernet cable between the phone and Cisco network switch. Visual inspection identified physical cable damage consistent with the timing of the failure. Repair of the Ethernet cable restored service.
